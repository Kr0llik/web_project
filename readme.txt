Проект представляет собой сервис подбора книг с учётом предпочтений пользователя. Пользователь может ответить на несколько вопросов для определения необходимого жанра. Так же пользователь может самостоятельно выбрать жанр произведений, которые хочет прочитать. Или может просто выбрать книгу для прочтения
